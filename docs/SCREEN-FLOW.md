## 全体フロー図

```
┌─────────────────────────────────────────────────────────────────┐
│  起動フロー                                                       │
│                                                                   │
│  [スプラッシュ]                                                    │
│       ↓ 自動遷移 (500ms)                                          │
│  [オンボーディング]                                                │
│       ↓ "はじめる" ボタン                                         │
│  [ログイン] ←──────────────── "アカウントをお持ちの方はこちら"    │
│       ↓ ログイン成功                  ↑                           │
│       └──── [新規登録] ──────────────┘                            │
│                  ↓ 登録成功                                        │
└─────────────────────────────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────────────────────────────┐
│  メイン (ボトムタブナビゲーション)                                 │
│                                                                   │
│  ┌────────┬──────────┬──────┬────────┬────────┐                  │
│  │ ホーム  │ ポイント  │  AI  │ ランク  │  設定  │                  │
│  └────────┴──────────┴──────┴────────┴────────┘                  │
└─────────────────────────────────────────────────────────────────┘
```

## タブ別 詳細遷移

### 🏠 ホームタブ (`app/(tabs)/home.tsx`)

```
[ホーム]
  ├── 🔔 通知ボタン → [通知履歴]
  │
  ├── 💰 StatCard「今日の獲得」→ [ポイントタブ]
  ├── 💰 StatCard「残高」→ [ポイントタブ]
  │
  ├── 🏆 RankProgressCard (タップ) → [ランクタブ]
  │
  ├── 📡 StatusCard「詳細 →」→ [参加状況詳細]
  │
  ├── 📶 StatCard「Wi-Fi」→ (表示のみ)
  ├── 🔋 StatCard「バッテリー」→ (表示のみ)
  ├── 🔗 「接続詳細 →」→ [接続詳細]
  │
  ├── 🤖 AIショートカット
  │       ├── チャット → [AIチャット]
  │       ├── 要約 → [文章要約]
  │       ├── 翻訳 → [翻訳]
  │       └── 生成 → [文章生成]
  │
  └── ⭐ サポーターバナー (未加入時) → [サポータープラン]
```

### 💰 ポイントタブ (`app/(tabs)/points.tsx`)  ★新規追加

```
[ポイント]
  ├── 残高カード（現在の残高・今日/今週/今月）
  ├── 総獲得 / 総使用 サマリー
  └── 取引履歴リスト（直近10件）
        → 外部遷移なし（タブ内完結）
```

### 🤖 AIタブ (`app/(tabs)/ai.tsx`)

```
[AIツール一覧]
  ├── AIチャット (30pt) → [AIチャット]
  ├── 文章要約 (20pt) → [文章要約]
  ├── 翻訳 (15pt) → [翻訳]
  └── 文章生成 (25pt) → [文章生成]
```

### 🏆 ランクタブ (`app/(tabs)/rank.tsx`)

```
[ランク]
  └── ランクカード・スコア・進捗バー
        → 外部遷移なし（タブ内完結）
```

### ⚙️ 設定タブ (`app/(tabs)/settings.tsx`)

```
[設定]
  ├── 参加設定 (Wi-Fiのみ・充電優先・省電力停止) → トグルスイッチ（画面内）
  │
  ├── 通知設定
  │       ├── プッシュ通知 → トグルスイッチ（画面内）
  │       └── 通知履歴 → [通知履歴]
  │
  ├── アカウント
  │       ├── サポータープラン → [サポータープラン]
  │       ├── ポイント履歴 → [ポイントタブ]
  │       └── デバイス情報 → [接続詳細]
  │
  ├── サポート・法的事項 → (外部リンク予定 / Phase 2)
  │
  └── ログアウト → アラート確認 → [ログイン画面]
```

## モーダル/スタック画面

```
[接続詳細]
```

```
[AIチャット]
```

```
[文章要約]
```

```
[翻訳]
  ├── 戻るボタン → 前の画面
  └── 翻訳ボタン → モック結果表示（画面内）
```

### ✏️ 文章生成 (`app/draft.tsx`)

```
[文章生成]
  ├── 戻るボタン → 前の画面
  └── 生成ボタン → モック結果表示（画面内）
```

### ⭐ サポータープラン (`app/supporter.tsx`)

```
[サポータープラン]
  ├── 戻るボタン → 前の画面

```

### 🔔 通知履歴 (`app/notifications.tsx`)

```
[通知履歴]
  └── 戻るボタン → 前の画面
```

### ❌ エラー/待機 (`app/error.tsx`)

```
[エラー画面]
  └── 再試行ボタン → 前の画面 / リロード
```
